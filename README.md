# first-project
新闻NLP分类识别系统，采用BERT模型，实现对新闻文本的主题（如科技、体育、财经）自动分类，包含数据预处理、模型训练与评估模块。

## 数据集说明
本项目使用 **THUCNews 新闻文本分类数据集**，该数据集由清华大学自然语言处理实验室整理，包含14个类别、74万条新闻数据。

### 数据集获取
由于数据集文件体积较大，未上传至本仓库。你可以通过以下方式获取：
1.  官方地址：[THUCNews 数据集](http://thuctc.thunlp.org/)
2.  开源镜像：可在 Kaggle、Github 等平台搜索 "THUCNews" 获取

### 数据集使用
下载后，将数据集解压至 `demo/data/` 目录下，即可运行 `dataset.py` 加载数据。
